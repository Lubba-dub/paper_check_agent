# 离线镜像包说明

本目录用于存放默认离线部署所需镜像归档。

为避免 GitHub 仓库体积过大，`.tar` 镜像文件不随仓库提交；需要离线部署时，请在实际交付介质中单独提供，并放回本目录。

当前离线路径默认需要以下镜像：

- `article-check:platform`
- `nginx:alpine`

导入方式：

```powershell
.\一键导入离线镜像.ps1
```

导入完成后，执行：

```powershell
.\一键离线部署.ps1
```

说明：

- 当前离线包默认不包含 `grobid/grobid`
- 如需 PDF 增强解析，请在有外网或私有镜像仓库的环境中额外准备 GROBID 镜像
